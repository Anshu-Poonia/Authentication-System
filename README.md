Authentication System using MERN Stack
--------------------------------------

In this project, we create a JWT authentication system where user register their account and login. 

Features of this project:
  - User Register itself as a new user.
  - Login after the registration.
  - Logout.
  - Reset the password, with the help of OTP.
  - Email verification using the OTP verify.
  - Create the APIs in the backend.
  - Create user interface in frontend using react.
